# EchoAI 工作区短视频功能区开发计划列表

## 0.1.0-短视频功能区定位与范围确认

- [x] 明确短视频模块属于 EchoAI 的一个独立功能区，而非独立产品
- [x] 明确该功能区在 EchoAI 中的入口位置、导航层级与工作区归属
- [x] 明确短视频功能区与 EchoAI 现有能力的复用边界
- [ ] 明确短视频功能区首期目标：脚本生成、剪辑思路、爆款文案提取、风格参考
- [ ] 明确短视频功能区不承担重型视频编辑职责，仅输出可执行内容方案
- [ ] 梳理短视频功能区与其他功能区的数据隔离与共享规则

## 0.2.0-现有工作区结构梳理与接入方案

- [ ] 梳理 EchoAI 当前工作区的页面结构、路由结构、布局系统
- [ ] 梳理 EchoAI 当前已有的公共组件、表单能力、编辑器能力、任务流能力
- [ ] 梳理 EchoAI 当前已有的项目管理、历史记录、用户配置、模板能力
- [ ] 评估短视频功能区应采用“独立页面”还是“工作区子页面”模式接入
- [ ] 设计短视频功能区与现有工作区 UI 风格保持一致的接入方案
- [ ] 设计短视频功能区的最小侵入式改造方案，避免影响现有功能区

## 0.3.0-短视频功能区信息架构设计

- [ ] 设计短视频功能区首页
- [ ] 设计短视频脚本生成页面
- [ ] 设计爆款文案拆解页面
- [ ] 设计风格参考/素材输入页面
- [ ] 设计剪辑思路输出页面
- [ ] 设计历史记录与结果管理页面
- [ ] 设计模板选择与结果复用入口
- [ ] 设计页面间跳转关系与操作闭环

## 0.4.0-短视频脚本生成功能

- [ ] 支持按主题生成短视频脚本
- [ ] 支持按平台区分生成策略：抖音、视频号
- [ ] 支持按时长生成：15 秒、30 秒、60 秒、90 秒
- [ ] 支持按内容类型生成：口播、剧情、解说、混剪文案
- [ ] 支持输出标题、开头钩子、正文、结尾 CTA
- [ ] 支持同一主题生成多个脚本版本
- [ ] 支持脚本结构化展示，而非单段纯文本返回

## 0.5.0-爆款文案提取与拆解功能

- [ ] 支持用户粘贴爆款文案进行分析
- [ ] 支持导入转录文本进行结构提取
- [ ] 支持提取钩子、冲突、观点推进、结尾转化结构
- [ ] 支持识别文案中的高频句式、金句、情绪词和节奏词
- [ ] 支持输出可复用的文案结构模板
- [ ] 支持将爆款结构一键应用到新的选题生成中

## 0.6.0-剪辑思路与分镜建议功能

- [ ] 支持根据生成脚本输出剪辑思路
- [ ] 支持输出分镜表：段落、旁白、镜头、字幕重点、建议时长
- [ ] 支持输出镜头类型建议：近景、口播、B-roll、转场插入等
- [ ] 支持输出字幕强化建议与重点词提示
- [ ] 支持输出背景音乐情绪建议与音效点建议
- [ ] 支持导出适合给剪辑执行的结构化结果

## 0.7.0-风格参考与素材喂养能力

- [ ] 支持上传文本、脚本、台词、历史内容作为参考素材
- [ ] 支持对参考素材进行风格分析
- [ ] 支持提取风格特征：语气、节奏、句式、情绪、常见表达
- [ ] 支持基于风格模板生成脚本
- [ ] 支持“参考某种角色气质/叙事方式”，避免高风险真人完全复刻表达
- [ ] 支持保存并复用风格模板
- [ ] 支持将风格模板接入短视频脚本生成流程

## 0.8.0-编辑器与交互优化

- [ ] 支持脚本结果的局部重写，而不是每次整篇重生成
- [ ] 支持对标题、开头、正文、结尾分别优化
- [ ] 支持“更口语化 / 更强钩子 / 更像视频号 / 更像抖音”的局部改写
- [ ] 支持多版本结果切换与对比
- [ ] 支持结果手动编辑后再次智能优化
- [ ] 支持按模块复制：标题、脚本、分镜、封面文案
- [ ] 支持导出为 Markdown、TXT、Word 或表格格式
